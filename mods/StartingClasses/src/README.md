# Source files for Starting Classes go here

The compiled `StartingClasses.dll` is in the parent directory. To rebuild from source, drop your `.cs` files here and run:

```bash
./build/build.sh StartingClasses
```

If you don't have source files yet, you can decompile the DLL with [ILSpy](https://github.com/icsharpcode/ILSpy) or [dnSpy](https://github.com/dnSpyEx/dnSpy) and check the result back in here.
